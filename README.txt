1.HOW TO COMPILE
1.1 install boost head files and libraries into 3rdparty/boost/include and 3rdparty/boost/lib
1.2 open msvc solution live.sln, and compie it
1.3 you will get bin/h264streamer.exe

2.HOW TO USE THE PROGRAM
just run get bin/h264streamer.exe, you can get a url on the terminal.
Now you can use ffplay to play the url.
